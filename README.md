# BancoDeDados-Seguradora
Script SQL para SQL Server que cria o banco de dados de uma seguradora com 6 tabelas (Cliente, Seguradora, Plano, Contrato, Apólice e Segurado), constraints de integridade, checks de data, inserção de dados de exemplo e consultas. Desenvolvido como atividade acadêmica (TP1 – 2º Bimestre).
